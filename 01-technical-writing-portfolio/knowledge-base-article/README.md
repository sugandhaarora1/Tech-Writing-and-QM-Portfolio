# Knowledge Base Article Sample

## Title
How to Troubleshoot Missing Approval Notifications

## Applies To
- Sonata Digital 5.4
- Approval workflow module
- Operations users

## Problem
A user does not receive an approval notification after submitting a transaction for review.

## Possible Causes
- Notification rule is disabled.
- User role does not include approval permissions.
- Transaction status is not eligible for approval.
- Email notification service is delayed.

## Resolution

### Step 1: Check Transaction Status
Confirm that the transaction status is **Pending Approval**.

### Step 2: Verify User Role
Check whether the approver has the correct approval role assigned.

### Step 3: Review Notification Settings
Confirm that approval notifications are enabled for the workflow.

### Step 4: Escalate if Required
If the issue continues, contact product support and include:
- Transaction ID
- User ID
- Timestamp
- Screenshot of workflow status

## Expected Result
The approver receives a notification once the transaction reaches the correct workflow state.
